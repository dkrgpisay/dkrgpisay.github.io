INFORMATION MONSTER
Client: Computer Society of the Ateneo

Information Monster is a web system that focuses on 3 key member related functionalities - (1) Registration, (2) Activity Tracking, and (3) Rewards and Achievements.

Different departments in the organization track different types of success indicators but basically stem from the same datasource which is member activity tracking. The project started off as an enhancemet of the already then developed Member Registration System which grew to a bigger project - now the Information Monster.
COMPANY OF ATENEO DANCERS
Client: Company of Ateneo Dancers

The Company of Ateneo Dancers website is a site designed by the organization's creatives team. The site functions as an informative site but has a unique functionality - a booking system. Furthermore, the site tries to feature the on-going event/competition of the organization by incorporating these entries with Google Maps.

The site is built using Drupal with highly customized themes and functionalities.
KAYA NATIN
Client: Kaya Natin (via Works of Heart)

The Kaya Natin website is a website developed on behalf of Works of Heart. The site is built using Wordpress. Notable features include a header slider, a featured section and page/content categorization.
PHOENIX TRANSITION ANALYTICS
Client: Hewlett-Packard

PTA is an internal web system for our team in Hewlett-Packard. The system is similar to a Project Management System but incorporates key performance indicators based on the team where the system is being implemented. It shows a high level view of the different projects handled by the organization, status and key contacts.

The system has a front-end data reporting UI that highlights information based on KPIs and a back-end data input UI for collecting project information from key project owners.
